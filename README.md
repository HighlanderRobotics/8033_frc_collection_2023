## Climbing Challenge

```dart
0 = No Climb
1 = Failed Climb
2 = Charged
3 = Supported
4 = Failed
5 = In Community
```

## Actions

```dart
0 = Picked up Cube
1 = Picked up Cone
3 = Placed Object
4 = Dropped Object (On Accident)
5 = Delivered to Team
6 = Blocked
7 = Pinned
8 = Entered Community
9 = Exited Community
```
